# Sygg - Project Overview

Sygg (See Your Goals Grow) is a wallpaper manager that shows a wallpaper that changes based on the user's daily goals tracking progress. The generated wallpaper shows a GitHub similar interface with contribution graph; where each cell represents a day and the color of the cell represents the progress of the user in their goals for that day. The wallpaper is updated when the user checks in their goals.

# Tech Stack

Web App: TypeScript, React, TailwindCSS, DaisyUI
Native Agent: Go
Cloud: Firebase
